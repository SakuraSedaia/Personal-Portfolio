Sakura's Character Rig

What is Sakura's Character Rig?

Sakura's Character Rig, aka SACR, is an advanced Minecraft rig designed and built inside Blender, it is designed to be usable for any purpose, while most versions up to now have been made with Animations in mind, but with the R5 release more detail oriented features such as Pupils and enhanced Materials help make the rig more controllable for Single frame renders.

Go to the download page for SACR

SACR Github

The SACR Github page features all versions of the Rig including the very first versions of SAMR, the predecessor for SACR. Though any version of the Rig prior to R4 is extremely unstable and buggy, as well as laggy.
