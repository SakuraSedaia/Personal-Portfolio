Sakura's Character Rig

What is SACR?

SACR is a Advanced Minecraft Rig designed and built inside Blender, it is designed to be usable for any purpose, while most versions up to now have been made with Animations in mind, but with the R5 release, the plan is to add in features to make it more Single-frame Render friendly.

SACR Github

The SACR Github page features all versions of the Rig including the very first versions of SAMR, the predecessor for SACR. Though any version of the Rig prior to R4 is extremely unstable and buggy, as well as laggy.
